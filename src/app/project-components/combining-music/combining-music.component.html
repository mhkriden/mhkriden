Overview

With the increased use of music streaming services, music sites have become more specific on the type of music they provide. For example, YouTube.com offers an excellent platform for undiscovered talent, but often can be a poor choice for popular studio quality albums. The aim of this project was to create a tool which could interweave results and playlists from a range of music streaming sites. With this, dynamic playlists can be created where music from different sites can be combined into one convenient location.

It was this project that first got me interested in web application development and won me best student project 2016 as voted by school students at the Swansea University project fair. This project was also the inspiration for my current master's project, which aims to turn the result of this project into a profitable music streaming application.

The application was primarily built using AngularJS and JavaScript on the front-end, along with Bootstrap for mobile-first styling. I also used a huge range of APIs during development including the YouTube and SoundCloud APIs. All of these APIs had to be handled with a common data source and micromanaged in a very unique way.
