Overview

The project's aim is a fully featured online music streaming service using the YouTube API for content. YouTube has a large community of amateur and professional musicians, but its focus is being a video sharing site, not a music streaming platform. The end goal of this will bring a sophisticated login system to allow users to save and seamlessly listen to different artists on the site.

The website also introduces a unique take on playlist management known as tags. Instead of being limited to creating playlists of songs, tags allow a user to seamlessly combine any album, artist, genre, or playlist together. This system also gives users heavy control over shuffling algorithms. Users can choose to even shuffle by randomly choosing artists instead of songs, and even set predefined weighted shuffling percentages.

The site is being built primarily using the Angular front-end framework, along with Redux for the website's extensive state-management needs, Auth0 for secure sessionless authentication, and ASP.NET Core for a reliable back-end API.
