Overview

When a patient ruptures their achilles tendon, there is a lengthy recovery process which takes place over several stages. Patients are instructed to perform daily stretches which are designed to aid the recovery process in a safe manner.

In order to help patients through this recovery process, my team and I are designing a mobile application which helps patients keep track of their recovery process. The app is very similar in design to a fitness application, helping patients keeps track of when and what exercises to perform, as well as providing statistics and vital medical information.

The mobile application is being built with the Ionic framework for a beautiful hybrid mobile experience targeting Android, iOS and desktop. We are also using an ASP.NET Core back-end for our API, hosted on Microsoft Azure.
